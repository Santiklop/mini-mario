# Mini Mario

A single-file HTML5 canvas Mario-style platformer. Open `mario.html` in a browser to play.

## Controls

- **← →** or **A / D** — move
- **Space**, **↑**, or **W** — jump (hold for higher jumps)
- **R** — restart

## Features

- Run & jump physics with variable jump height
- Stompable goombas with cliff detection
- Coins (some hidden inside `?` blocks)
- Breakable bricks and pipes
- Scrolling camera with parallax clouds and hills
- Lives, death, respawn, and a flag at the end of the level

Built with vanilla JavaScript and the HTML5 canvas — no dependencies, no build step.
